## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/AbhilashMadi/geektrust-Challenges.git
   ```

2. Installing the dependencies:

   ```
   npm install
   ```

3. Starting the local server:

   ```
   npm start
   ```
